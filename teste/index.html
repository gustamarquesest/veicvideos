Hero · Posicionamento / UX
Isotipo SVG fantasma ocupa metade do hero — metade do espaço mais valioso do site é decoração vazia
Alta
Solução
Remover o .hero-bg-isotipo. Reestruturar o hero em layout split grid-template-columns: 1fr 1fr. Coluna esquerda: conteúdo de texto atual. Coluna direita: container .hero-visual com fundo #1a1200 e mensagem visual de placeholder até o vídeo de bastidores estar disponível. Impacto esperado
O hero passa de página de texto com decoração CSS para layout cinematográfico de dois campos. A estrutura visual de "produtora" é comunicada pelo próprio layout antes de qualquer imagem nova. Remove a sensação de vazio imediata que o visitante sente hoje.
Dimensões impactadas
  • Posicionamento — layout split é linguagem visual de produtoras, não de consultoras
  • Mobile first — no mobile o split vira vídeo-first (coluna visual no topo)
  • Percepção de autoridade — estrutura profissional antes de qualquer imagem nova
Esforço
1–2h de CSS · Sem dependências externas

Média

Ritmo visual geral · Direção de arte
8 de 9 seções têm fundo #131210 — o escuro perdeu poder dramático por ser o estado padrão, não a exceção
Alta
Solução
Implementar a regra dos três escuros: nunca três seções com fundo escuro puro consecutivas sem ruptura visual. Ajustes concretos:

— Seção Projetos Audiovisuais: fundo muda de #131210 puro para #1a1a16 com textura de grain CSS (box-shadow inset com noise muito sutil, 2% opacidade) — cria diferenciação sem mudar a paleta
— Seção Backstage: fundo muda para #0f0e0c (mais escuro que o padrão) — cria hierarquia entre seções escuras
— Seção Processo: fundo #1e1c1a (levemente mais claro)
Impacto esperado
O scroll pelo site passa a ter ritmo: o visitante sente alternância visual mesmo dentro da paleta escura. Cada seção tem identidade própria. A seção amarela do Propósito e a seção cream do FAQ se tornam mais dramáticas porque o escuro ao redor delas varia — não é mais todo igual.
Dimensões impactadas
  • Direção de arte — contraste entre seções é princípio fundamental de composição cinematográfica
  • Storytelling visual — o ritmo escuro/claro/escuro cria tensão e resolução ao longo do scroll
  • Percepção de autoridade — sites de produtoras de referência usam escuros variados, não um preto uniforme
Esforço
1h de CSS — ajuste de variáveis de fundo por seção
FAQ · Escaneabilidade
A resposta mais longa tem 4 subtópicos em uppercase amarelo + listas — formato dificulta escaneamento rápido
Média
Solução
Substituir os 4 subtópicos uppercase por ícones Tabler de 16px à esquerda de cada subtópico. Ícones sugeridos: ti-globe (presença), ti-chart-bar (comercial), ti-layout (conteúdo), ti-building (interno). O conteúdo das listas permanece idêntico — só o marcador visual muda.
Impacto esperado
Escaneamento ~40% mais rápido. Os ícones criam ancoragem visual que guia o olho entre os subtópicos sem exigir leitura sequencial. Melhora a experiência de quem busca resposta específica.
Esforço
20 minutos de HTML

Alta — maior impacto de conversão
Solução
Para cada uma das 8–10 necessidades listadas no seletor esquerdo: adicionar no topo do painel direito um frame 16:9 do portfólio correspondente, com badge "Exemplo real ▶" clicável que abre o modal de vídeo existente.

Mapeamento: Atrair clientes → Betim Futebol / Anúncio · Presença digital → Landing Page · Posicionamento → Cineart · Vendas → Biologistica · Fidelização → Meet Tecnologia · Comunicação interna → Treinamento NR · Eventos → Mangalarga · Cada frame: screenshot do vídeo existente, 800×450px mínimo.
Impacto esperado
Quando o visitante seleciona "Queremos vídeos para eventos" e vê imediatamente um frame real de evento filmado pela Veic, a decisão de entrar em contato se torna muito mais fácil. A seção passa de diagnóstico textual para portfólio navegável por necessidade. Esta é a mudança de maior ROI de conversão do site.
Redução de texto consequente
Com o frame visual presente, a descrição de cada solução pode ser reduzida de 2–3 frases para 1 frase. Atual: "Estrutura da sua empresa com vídeos que apresentam quem você é, o que faz e por que escolher seu negócio." → Proposta: "Vídeos que apresentam sua empresa antes da primeira reunião." O frame mostra o resultado; a frase nomeia o objetivo.

Briefing fotográfico
5 fotos de bastidores em 1 dia de shoot:

1. Câmera no tripé com lente em foco, luz de tungstênio ao fundo — formato 3:4 (slot grande). Câmera deve ser reconhecível como profissional.

4. Reunião de briefing — 2–3 pessoas, caderno/laptop, olhando para o mesmo ponto.
5. Editor na timeline do Premiere/DaVinci — tela reconhecível com frames.

Todas em ambientes reais da Veic. Sem estúdio neutro.
Impacto esperado
Esta é a seção que mais transforma a percepção de marca do site inteiro. Ver 5 fotos reais de câmera, set e equipe resolve o problema central do diagnóstico: "parece consultoria, não produtora". Nenhum texto consegue o que 5 fotos fazem em 2 segundos de scroll.

Princípio que orienta todo o plano O problema central do site não é estético — é de gênero. O site está visualmente competente para uma consultoria de comunicação. Para uma produtora audiovisual, falta o material bruto que define o gênero: câmera, luz, set, rosto humano. A Fase 1 corrige estrutura e copy. A Fase 2 usa o que já existe para comunicar visualmente.
